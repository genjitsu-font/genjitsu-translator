# Genjitsu Translator

現実の文字を幻日の文字に変換して表示します

## Development

### Run server

```
npm install
npm run dev
```

### Build

```
npm run build
```

## Create sprite

`glyph/` ディレクトリ以下に、 `あ.png` など文字名を含むファイル名で画像を配置してください。

```
python3 sprite.py
```
